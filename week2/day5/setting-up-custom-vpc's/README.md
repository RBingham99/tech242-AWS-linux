# Set up a VPC

2) Create subnets
3) Create an internet gateway
4) Attach the internet gateway to your VPC
5) Create route table
6) Associate route table with subnet
7) Add internet gateway to the route
8) Check VPC setup
9) Setup DB VM
10) Setup API VM
